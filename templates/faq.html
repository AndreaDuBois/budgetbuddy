Frequently Asked Questions

Getting Started

How do I set up Pocket Money for the first time?

Go to the Sign In page and click Create Account. You'll become the primary parent. From there, go to Events to create your first shopping event, then Members to add your kids.

How do I add my kids?

Go to Members → Add Member. Enter your child's name and a login email. You can set a password for them, or use Send Invite Link on their profile page so they can set their own password.

How do I add another parent or guardian?

From the Dashboard, tap Invite Co-Parent. A shareable link is generated — copy it and send it to them however you like (text, email, etc.). They click the link and create their own login. Co-parents have the same full access as you.

Shopping Events & Budgets

What is a Shopping Event?

An event is a named shopping trip — like "Back to School 2025" or "Holiday Gifts." Inside each event you create categories (Shoes, Shirts, Backpack, etc.) and set a dollar budget per category for each child.

Can I have more than one event at a time?

Yes. Multiple events can be active at the same time. Each kid sees budgets for all active events.

What happens when a budget is overspent?

The progress bar turns red and the app shows how much over budget the category is. You'll also see a budget advice notice. Nothing is blocked — it's informational so you can decide how to handle it.

Can kids adjust their own budgets?

Only if you allow it. On each child's profile, there's a toggle: Allow this member to adjust their own category budgets. By default it's off.

Receipts & AI Scanning

How does receipt scanning work?

When a kid (or parent) uploads a receipt photo or PDF, the AI reads it and pulls out each line item along with prices. It then suggests which budget category each item belongs to. You review and confirm before any totals are updated.

What are the four ways to upload a receipt?

  • Photo — snap or upload an image of the receipt
  • PDF — upload a digital receipt saved as a PDF
  • Paste text — copy the text from an email receipt and paste it in
  • Manual entry — type items and prices in yourself (no AI needed)

The AI got something wrong on my receipt. What do I do?

On the review screen you can edit each line item — change the description, price, or category — before confirming. Nothing is final until you tap Confirm.

Can parents upload receipts on behalf of a child?

Yes. From the parent receipt view, you can upload a receipt and assign it to any child in any active event.

Accounts & Privacy

Can kids see each other's budgets?

No. Each child's login only shows their own budgets and receipts. Only parents can see everything.

What if a kid forgets their password?

A parent can go to Members → [child's name] → Edit and set a new password, or generate a new invite link so the child can choose their own.

Where is my data stored?

All data is stored locally on the device or server where Pocket Money is running. Nothing is sent to any third-party service except receipt images/text, which are temporarily sent to the AI for reading.
